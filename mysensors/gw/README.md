**Description:**

This is a Mysensors serial GW using **RFM69W** radio module and **CP2102** usb to serial converter. 

**Key features:**

 - **USB to serial converter** using CP2102 specialized IC
 - **ATSHA204A** for signing
 - **MySensors** ready(Atmega328p + RFM69W)
